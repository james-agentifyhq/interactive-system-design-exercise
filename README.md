# System Design Study Kit

An interactive system design study platform powered by [Claude Code](https://claude.ai/claude-code). Clone this repo, open it in Claude Code, and use the slash commands below to practice.

## Getting Started

1. Clone the repo and open it in Claude Code
2. Enable voice dictation for a hands-free interview experience: https://code.claude.com/docs/en/voice-dictation
3. Use a tool like [Excalidraw](https://excalidraw.com) to draw architecture diagrams — you can append screenshots to your answers during mock interview sessions

## Slash Commands

- `/sdi-mock` — Run a Socratic system design mock interview with scoring and a saved transcript
- `/sdi-flashcard` — Run a spaced-repetition flashcard session that tracks your progress with SM-2 scheduling
